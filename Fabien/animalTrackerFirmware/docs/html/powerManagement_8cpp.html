void configureTimer ( )

Initialiser la connexion LoRaWAN via OTAA pour TTN.

Permet de réaliser la première communication entre la carte et le réseau TheThingsNetwork, permet le paramètrage de la carte et l'authentification de la carte auprès de TTN.
