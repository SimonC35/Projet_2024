Envoyer les données via le module Lo\+Ra de la carte. 

Sollicite le module Lo\+Ra pour envoyer les données GPS acquises. La fonction reçoit un tableau de données et sa taille, puis tente de les envoyés pendant retry\+Count tentative(s).

\begin{DoxyNote}{Note}
La fonction est non bloquante, en cas de réussite ou d\textquotesingle{}échec le système passera au cycle suivant. 
\end{DoxyNote}
